Description

This prefab allows you to add a Healing Power Up in Unity. It includes essential components for increasing the players’ health.

Components

  • Game Item Script
    • Base for the power up.
    • Can set the id, itemName, description, isAvailable, and itemSprite.
    • Debug logs when item is available and when item is collided with.
  • Power Up Script
    • Base classes for the power up.
    • Debug logs when powerup has been applied to player.
    • Can modify the values for the healing item.
  • Health Boost Script
    • Main script for applying health boost.
    • When a player collides with the object it increases their health.
  • Sprite Renderer
    • Displays the healing sprite.
    • Customize the healing sprite with the Sprite Renderer component.
    • Ensure that the sorting layer is appropriate to display the healing sprite.
  • Box Collider 2D
    • Provides a collision boundary around the healing item.
    • Set the dimensions to match the size of the sprite for accurate collision.
  • Rigidbody 2D
    • Manages the sprite’s physics.

Setup Instructions

Drag the Healing Item prefab into the scene. Ensure the Game Item, Power Up, and Health Boost scripts are configured with appropriate settings for healing values. Adjust the Box Collider 2D appropriately to ensure accurate collision detection. Test the Healing item in play mode to confirm behavior.

Requirements

Unity 6000.0.36f1 or later
